    Specify whether Yieldprobe should use Geolocation data.

    If false, Yieldprobe will not try to access geolocation information, even if it is available to the application. If true, Yieldprobe will try to access geolocation information only if the app already has access to it. Yieldprobe will not cause location permission prompts in your application.
